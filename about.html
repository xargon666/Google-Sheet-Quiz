About

There are a lot of quiz apps out there, but I've yet to find a free-tier app with the ability to pull data from spreadsheet for quickly building an exam style quiz.

After some research, I found that Googlesheets has a "Form Builder for Google Sheets™" extension that allows you to do just that... but it limits the user to about 100 questions a month.

And I took that personally.

It occured to me that somebody had probably created a solution for this already and posted it to Github, so I went looking and quickly discovered subhranshuchoudhury's Google-Sheet-Quiz Project.

. The vanilla js project struck me as a great tool for students looking to learn new skills and practice for exams, and it inspired me to build on it's features.

Vanilla JS is bad, but it has a few beginner friendly advantages:

  • Download and launch the site immediately, without any repacking/compiling
  • Non-minified live code is exposed to the user
  • It's great coding practice!

I decided to rebuild the site from the ground up, replacing code, styling, and exploring ideas I've been interested to try.

I replaced hardcoded links to Googlesheets with a cool feature: saving data into localStorage. This feature allows users to add their own quizes with much less work, and return to the saved quiz (from the same device) without the need for backend hosting or user sign in.

The process for adding a new quiz is a little tricky... but it's free!

Visit the github page for instructions and updates on this on-going project.
