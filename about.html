About

I discovered subhranshuchoudhury's Google-Sheet-Quiz Project after reaching the export limit in the "Form Builder for Google Sheets™" extension.

It seemed like a quick student project for learning HTML/CSS/JS and it did the job well.

This project struck me as a useful tool for students looking to learn new skills and practice for exams, and I could see it's potential.

I decided to rebuild the site from the ground up, replacing the code, improving functionality, styling and the user experience.

I extended the functionality to save data into localStorage. This feature lets users to add their own quiz data to the site with less work, and access and access the saved quiz from the same device later, without backend hosting.

It still requires some setup...
Full instructions can be found on my github page.
